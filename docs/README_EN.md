# TextCleaner

A Spanish OCR text-cleaning tool, mainly designed to clean Markdown text generated by MinerU, OCR software, or PDF parsing tools.

Current version: v1.2.0

## Features

TextCleaner is designed for Spanish academic texts, reports, books, and other OCR outputs. It helps users automatically perform common text-cleaning tasks, such as:

* Removing extra blank lines, abnormal spaces, and formatting noise after OCR
* Repairing Spanish word breaks:
  * Line-break hyphenation, for example: situa-ción → situación
  * Line breaks without hyphens, for example: situa\nción → situación
  * In-line word breaks
* Restoring some missing Spanish accents, for example: politica → política
* Removing footnote numbers and annotation numbers
* Removing formula blocks and formula-related noise
* Removing front covers, copyright pages, and tables of contents from books or reports
* Removing references, Bibliografía, Notas, appendices, advertisements, subscription pages, and other non-body content
* Supporting drag-and-drop cleaning of Markdown files
* Supporting batch processing of Markdown files
* PDF-assisted proofreading
* Splitting large PDF files
* Counting tokens/word forms

## Features Not Yet Available

* Automatic batch calling of MinerU to parse PDF files

This feature currently has known issues. In the current version, it is recommended to manually parse PDFs with MinerU first, and then place the generated Markdown files into `output/mineru_raw/` for cleaning.

## Use Cases

This tool is suitable for:

* Cleaning Spanish PDF texts after OCR
* Post-processing MinerU parsing results
* Preprocessing Markdown files before corpus construction
* Batch cleaning of academic texts, industry reports, books, and other documents

## Installation and Usage

For the first installation, extract the only ZIP file in the project folder into the program directory.

- macOS: Run `1_首次安装_macOS.command` for the first installation, then drag files onto `TextCleaner.app`.
- Windows: Run `1_首次安装_Windows.bat` for the first installation, then drag files onto `2_拖放清洗_Windows.bat`.

Run from source:

```bash
python3 -m pip install -r requirements.txt
python3 "textcleaner V1.2.py"
```

## Common Entry Points

- Double-click `拆分大PDF.command`: split PDF files over 250 pages into 200-page volumes and save them to `pdf_split/` on the Desktop.
- Double-click `TextCleaner.app`: clean dragged Markdown files.
- Run `textcleaner V1.2.py`: clean Markdown files in `output/mineru_raw/`.

## Directory Structure

- `data/`: stop phrases and candidate words
- `docs/`: project documentation and usage guides
- `tests/`: automated tests
- `output/`: input, output, and logs generated by TextCleaner
